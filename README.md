# PassTool
PassTool is a Python application that generates secure passwords using various algorithms. It offers three types of password generation: PIN generation, random password generation, and memorable password generation using words from the NLTK corpus.
## Features
- **PIN Generation**: Create numeric PINs of specified length.
- **Random Password Generation**: Generate complex passwords with options for length, symbols, and numbers.
- **Memorable Password Generation**: Generate passwords using random words from the NLTK corpus, with options for capitalization and separators.
## Requirements
- Python 3.6 or higher
- NLTK library
- Streamlit library
